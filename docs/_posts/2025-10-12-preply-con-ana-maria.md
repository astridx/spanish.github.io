## Verbos como gustar

El sujeto es **lo que gusta**, y la persona a la que le gusta se pone en **dativo (me, te, le, nos, os, les)**.

**Me gusta el libro.** → *Mir gefällt das Buch.*
**Te gustan los gatos.** → *Dir gefallen die Katzen.*


| Verb                | Bedeutung                          | Beispiel                        |
| ------------------- | ---------------------------------- | ------------------------------- |
| **encantar**        | lieben, sehr mögen                 | Me encanta el chocolate.        |
| **interesar**       | interessieren                      | Nos interesa la historia.       |
| **molestar**        | stören                             | Le molesta el ruido.            |
| **importar**        | wichtig sein                       | ¿Te importa si abro la ventana? |
| **fascinar**        | faszinieren                        | Me fascina el arte moderno.     |
| **aburrir**         | langweilen                         | Me aburre la clase.             |
| **doler (o → ue)**  | weh tun                            | Me duele la cabeza.             |
| **faltar**          | fehlen                             | Nos falta dinero.               |
| **quedar**          | übrig bleiben, passen              | Me queda bien la camisa.        |
| **parecer**         | scheinen                           | Me parece una buena idea.       |
| **interesar**       | interessieren                      | Te interesa la música clásica.  |
| **chocar**          | total nerven / gar nicht mögen     | Me choca su actitud.            |
| **caer bien / mal** | sympathisch / unsympathisch finden | Me cae bien tu hermano.         |
| **hacer falta**     | nötig sein / brauchen              | Me hace falta dormir.           |
| **tocar**           | dran sein / zufallen               | Te toca limpiar la cocina.      |
| **sobrar**          | übrig bleiben                      | Nos sobra comida.               |
| **abrumar**         | übrig bleiben                      | Me abruma la situación.         |



| Person              | Pronomen |
| ------------------- | -------- |
| yo                  | me       |
| tú                  | te       |
| él/ella/usted       | le       |
| nosotros            | nos      |
| vosotros            | os       |
| ellos/ellas/ustedes | les      |

**Combinar con amplificadores**

| ---------- | ------------------------ |
| un poco    | ein bisschen             |
| bastante   | ziemlich                 |
| mucho      | sehr                     |
| muchísimo  | total / überaus          |
| demasiado  | zu sehr                  |
| nada       | gar nicht (bei Negation) |


* Me gusta muchísimo leer.
* No me gusta nada el invierno.
* Me interesa bastante la política.
* Me gusta mucho la música clásica.
* No me gusta nada madrugar.
* Me gusta un poco el café.

## **Positivo**

| Deutsch                 | Spanisch                                  | Wortwörtlich / Hinweis             |
| ----------------------- | ----------------------------------------- | ---------------------------------- |
| Das mag ich auch        | A mí también me gusta / A mí también      | „Mir gefällt es auch“              |
| Ich auch                | Yo también                                | Kurzform, Kontext klar             |
| Geht mir genauso        | A mí también                              | Wird wie „mir auch“ benutzt        |
| Ich liebe es auch       | Yo también lo/la/los/las amo / me encanta | Je nach Objekt                     |
| Ich finde das auch toll | Yo también lo/la/los/las encuentro genial | Weniger gebräuchlich, aber korrekt |


## **Negativo**

| Deutsch                      | Spanisch                                | Hinweis                     |
| ---------------------------- | --------------------------------------- | --------------------------- |
| Das mag ich nicht            | A mí tampoco me gusta                   | „Mir gefällt es auch nicht“ |
| Ich auch nicht               | Yo tampoco                              | Kurzform                    |
| Geht mir genauso (negativ)   | A mí tampoco                            | Bei Negativaussagen         |
| Ich finde das auch nicht gut | Yo tampoco lo/la/los/las encuentro bien | Weniger gebräuchlich        |
